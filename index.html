Jejak Orangutan Website

Jejak Orangutan is an educational website built using HTML, CSS, and JavaScript with the goal of increasing awareness about orangutan conservation and environmental sustainability. The project presents information through a structured and engaging interface, making it easier for users to learn about the challenges faced by orangutans and the importance of protecting their natural habitat. During development, I focused on creating a responsive and accessible experience while maintaining a clean visual design. This project allowed me to practice front-end development skills, content organization, and user experience design while working on a meaningful environmental topic.
